# vue-toast

## 导入组件库
```
import Component from '../packages/index';
```

## 注册组件库
```
Vue.use(Component.Toast);
```

## git 地址
```
https://github.com/cwy930618cwy/cwy-toast
```

## 使用方法
```
this.$toast.show({
  showType: '', // 显示图标类型 'success'
  toastMessage: '' // 显示的文字
})
```
